## 群

* 封閉性： a * b in G
* 結合性： (a * b) * c = a * (b * c)
* 單位元素： a * I = a   // 在乘法運算中 I=1, 加法運算中 I=0
* 反元素： a * ia = I    // ia 為 a 的反元素，也是 Ｇ 的成員

具備『交換性』的群稱為『交換群』或『阿貝爾群』（Abelian group）
